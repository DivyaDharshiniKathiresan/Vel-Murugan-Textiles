## Deploying to GitHub Pages
1. Push all files to this repository
2. Go to **Settings → Pages**
3. Source: `main` branch, root folder → **Save**
4. Site goes live at: `https://divyadharshini.github.io/new-velmurugan-textiles/`

## Google Search Console Setup
1. Visit [search.google.com/search-console](https://search.google.com/search-console)
2. Sign in with `divyadharshini.k003@gmail.com`
3. Add property → URL prefix → enter the GitHub Pages URL
4. Verify → Submit sitemap: `sitemap.xml`
